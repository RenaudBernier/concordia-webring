A webring aggregating personal websites/portfolios created by students at Concordia University. 

## Adding your Website

To add your website to the Concordia Webring:

1. Fork this repository;
2. Add your info to `public/webring.json` according to the format below;
```json
{
  "name": "John Doe",
  "website": "https://JohnDoe.com",
  "year": 2067,
  "program": "COMP"
}
```
3. Create a pull request to this repository.

Or, you can fill out [this Google Form](https://forms.gle/u6oJUh1wRof1Lxcy7) and a maintainer will add your website for you.
